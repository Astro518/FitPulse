# 🔥 FitPulse - Gamified Personal Fitness Tracker

FitPulse transforms fitness tracking into an engaging game-like experience. Log your daily activities, earn experience points, level up, and unlock achievement badges as you crush your health goals.

## ✨ Features

### 🎮 Gamification System
- **Level Up** - Earn XP from workouts and meals, watch your level grow
- **Achievement Badges** - Unlock "Step Master", "Hydration Hero", and "7-Day Streak" badges
- **Motivational Quotes** - Daily inspiration to keep you going

### 🏃‍♂️ Activity Tracking
- Step counter with daily goal (10,000 steps)
- Workout logging with calorie tracking
- Workout history for today's activities

### 🥗 Nutrition Management
- Meal logging with calorie tracking
- Water intake tracker (goal: 8 glasses/day)
- Meal history display

### 📓 Fitness Journal
- Write daily entries with mood selection
- Tag entries (Workout, Nutrition, Progress, Goals, Motivation)
- Search and filter journal entries
- Emoji mood indicators

### 👤 Profile & Stats
- BMI calculator
- Personal fitness goals (Weight Loss, Muscle Gain, Maintenance)
- Dark/Light mode toggle
- Local storage persistence - your data stays with you

## 🛠️ Technologies Used
- HTML5
- CSS3 (CSS Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Font Awesome Icons
- Local Storage API

## 🚀 How to Use
1. Log your steps using + / - buttons
2. Add workouts and meals with calorie tracking
3. Drink water by clicking "Add Glass"
4. Write journal entries about your fitness journey
5. Watch your level rise and badges unlock!
